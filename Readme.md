ABSTRAK
Perpustakaan merupakan salah tempat yang sering dikujungi oleh kalangan pelajar baik itu siswa maupun mahasiswa. Keamanan suatu perpustakaan dapat diwujudkan dengan di sediakannya tempat penyimpanan barang yaitu sebuah loker. Ketika suatu loker masih dibuka secara manual dan peminjaman loker secara manual, maka masih sering terjadi padatnya pelayanan oleh pustakawan. Selain itu sering lupa nya pengguna untuk menutup kembali pintu loker yang merupakan suatu kelalaian pengguna sehingga menyebabkan kehilangan barangnya. Sistem penyimpanan loker berbasis IoT dapat dirancang sebagai sistem yang dapat membuka loker menggunakan KTP-el tanpa perlu meminjam kunci loker dan mencatat peminjaman loker. Sistem ini dibuat dengan menggunakan modul RFID MFRC522 yang berfungsi untuk membuka loker menggunakan KTP-el sekaligus mencatat penggunaan loker secara otomatis, dan juga sensor switch magnetic mc38 yang berfungsi mendeteksi pintu loker yang terbuka atau tertutup. RFID MFRC522 membaca KTP-el dengan maksimal jarak 1 cm. Setiap jenis kartu memiliki jarak baca yang berbeda-beda. Sensor switch magnetic mc38 mendeteksi pintu dengan jarak maksimal 1 cm, ketika sensor terdeteksi maka status pintu tertutup dan menampilkan status data di halaman website. Data sensor tersebut dikirim menuju basis data MySQL menggunakan mikrokontroler NodeMcu. Semua data pada MySQL ditampilkan pada halaman website yang telah dibuat menggunakan Php Native. Data ditampilkan dalam bentuk teks, tabel, grafik, dan dalam waktu nyata.
Kata kunci: NodeMcu, RFID, Sensor magnetic mc38, Php Native.
